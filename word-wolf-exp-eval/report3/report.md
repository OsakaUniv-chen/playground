※ **Yes率**は「ロボットの動作は人間による遠隔操作だと思うか」への「はい」の割合。**PTL（Perceived Teleoperation-Likeness）**は、この回答と回答への自信度（低い・普通・高い）を統合して0–100%へ写像した指標（50%が是非の境界）：はい・高い=100%／はい・普通=80%／はい・低い=60%／いいえ・低い=40%／いいえ・普通=20%／いいえ・高い=0%（report2 §5.2と同じ定義）。

## 2. TELEの行動系列のランダム性検証

**方法**：Wald–Wolfowitzの游程検定（runs test）。tick系列（4Hz）の`tel_side`（左/右）について、観測游程数を「同じ左右比率のランダム順列」での期待游程数と比較し、正規近似のzで両側検定する（零仮説の左右比率はTELE自身の観測データから取り、実験のRandom方策の生成過程とは無関係）。複数局・複数組を束ねる場合は各局のR_obs・期待値・分散を合算してから統合zを算出する。`tel_side`はTeleモードに限らず全52局（13組×4モード）に存在するため、以下4区分で検定する：

- **2.1** 13組・全部データ（52局、4モード）
- **2.2** 13組・Teleモードのみ（13局）
- **2.3** group1（G1・G6）のTeleモード
- **2.4** group2（9組）のTeleモード

**まとめ**：いずれの区分でも p<.001 でランダム性を明確に棄却する。

## 3. 被注視後の発話開始行動分析

**目的**：ロボットが誰かの方を向いた（注視した）ことが、実際にその人の発話開始を促しているかを、Tele・PSSP・DoA・Randomの全モードで確認する。

**方法**：ロボットの実際の向きが新しい側へ切り替わり、その後3秒間（12tick）向きが変わらなかったイベントを対象とする。さらに、切り替わった瞬間に注視された側の人がまだ発話していなかった（`gt_label` ≠ 注視側）場合に限定する。結果指標は、そのイベント後3秒以内に注視された側の人が発話状態になる（`gt_label` = 注視側）確率。対照として、同じイベント・同じ窓（切り替わり直後のtickから12tick先まで）で「対側（見られなかった人）」が発話状態になる確率も計算する——分母（イベント集合）は共通で、見る対象（注視側／対側）だけを入れ替えた指標なので、両者は互いに排他的ではない（どちらも発話する／どちらも発話しない、もありうる）。

**結果**：

n列の内訳は、イベント時（t0）の`gt_label`の内訳（Teleoperator:Others:対側、対側＝その時点で対側がすでに発話中だったケース）。

| モード | n（イベント数、内訳） | 注視された側 | 対側（対照） |
|---|---|---|---|
| Tele | 121（Teleoperator:Others:対側=41%:22%:36%） | 67.8% | 70.2% |
| PSSP | 140（Teleoperator:Others:対側=57%:30%:13%） | 68.6% | 39.3% |
| DoA | 118（Teleoperator:Others:対側=62%:27%:11%） | 62.7% | 38.1% |
| Random | 189（Teleoperator:Others:対側=38%:26%:36%） | 58.2% | 66.7% |
| **全部** | 568（Teleoperator:Others:対側=48%:27%:25%） | 63.7% | 54.8% |

注視された側の発話開始確率は PSSP(68.6%) > Tele(67.8%) > DoA(62.7%) > Random(58.2%) で、モード間の差は小さい。13組を対応ブロックとしたFriedman検定（組ごとの確率を4モードで反復測定として扱う）では有意差なし（χ²=2.35, df=3, p=.50）。

一方、PSSP・DoAは注視側が対側より大きく高い（68.6% vs 39.3%／62.7% vs 38.1%）のに対し、Tele・Randomはむしろ対側の方が高い（67.8% vs 70.2%／58.2% vs 66.7%）。この違いはn列の内訳と対応している：PSSP・DoAはイベント時に対側がすでに発話中だったケースが11〜13%しかないのに対し、Tele・Randomは36%と多い。対側がすでに発話中のケースが多いモードほど、その後の窓で「対側が発話状態」を記録しやすくなる（単なるターンの継続）ため、Tele・Randomの対側優位はこの内訳の違いで説明できる可能性が高く、注視の効果とは言いにくい。

## 4. 頭部yaw偏移と印象評価の相関分析

**目的**：§1でTeleの左右比率が組によって大きく偏っていることが分かった（§1.1）。この偏りは遠隔操作者の座り方・カメラとの相対位置が組ごとに統一されていなかったことに由来する可能性があり、Teleの主観評価が一部の組の強い右偏りによって系統的に押し下げられていないかを確認する。§1では偏りの小さい2組（G1・G6）への限定という二値的な比較だったが、ここでは組ごとの偏りの程度（連続量）とアンケート得点の相関を見ることで、より直接的に量的な関係を確認する。

**方法**：組ごとのTele局における遠隔操作者の頭部yaw平均（degree）と、その組でのPTL・GQS5尺度・GEQ5尺度の組平均値を、13組を単位にSpearman順位相関で対応づける。

**結果**：

| 指標 | rho | p |
|---|---|---|
| PTL | -0.19 | .539 |
| Anthropomorphism | -0.27 | .371 |
| Animacy | -0.29 | .345 |
| **Likeability** | **0.58** | **.037** |
| PerceivedIntelligence | -0.14 | .658 |
| PerceivedSafety | 0.35 | .248 |
| Flow | -0.25 | .412 |
| Competence | 0.14 | .640 |
| PositiveAffect | 0.07 | .822 |
| NegativeAffect | -0.42 | .157 |
| Tension | -0.02 | .943 |

（yawは正=左・負=右。rho>0は「より左寄り（＝より均衡）な組ほどスコアが高い」、すなわち右偏りが強いほどスコアが低い方向を意味する。）

11指標のうち有意なのはLikeability（rho=.58, p=.037）のみ。PTLを含む他の10指標はいずれも|rho|<.45で有意水準に達しない。

**まとめ**：

- 遠隔操作者の頭部yawが右に偏っている組ほど、参加者はTeleロボットを**好ましく思わない**（Likeability低下）傾向が、13組を通じた連続的な相関として確認された。
- 一方PTL（人間操作だと思う度合い）はyawの偏りとほぼ無相関。
- 他9指標（NegativeAffectを含む）は有意な相関を示さなかった。
- 以上より、「座位・頭向の統制不足がTeleの主観評価を系統的に押し下げた」という懸念は、少なくともLikeabilityについては支持される。
